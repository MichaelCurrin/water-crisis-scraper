# Water Crisis
>Scrape and explore data related to Cape Town's water crisis


## Data sources


### Dam Levels

A CSV of dam levels from 2012 to 2018 is obtainable at no cost from the [Dam levels data set](https://web1.capetown.gov.za/web1/opendataportal/DatasetDetail?DatasetName=Dam+levels) page, hosted on the _City of Cape Town Open Data Portal_. This is used in the [damLevels](waterCrisis/damLevels) section of the repo.
